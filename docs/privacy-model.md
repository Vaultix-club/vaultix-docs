# Privacy Model

Vaultix positions privacy as a product requirement rather than a marketing afterthought.

- Aggregate routing decisions should not broadcast the full strategy graph.
- Counterparty selection and liquidity sourcing should remain scoped to the minimum required participants.
- Connector adapters expose only the contract surface needed for execution and reporting.
- Public dashboards use reference market data and avoid implying that every displayed asset is live on Canton.
- Security review must verify that privacy claims in public docs do not exceed the actual deployment guarantees.
- Release materials should explicitly separate reference analytics from production routing behavior.
